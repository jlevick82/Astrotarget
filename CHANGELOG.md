# AstroTarget Changelog

## v0.9.0 – Field Ready (2025-08-28)
- Added **deduplication** across Messier, Caldwell, Bright NGC catalogs
- New **GPS clock** (updates live)
- Added **Moon phase + altitude + dark sky rating**
- New **Weather overlay** (with offline fallback message)
- New **Light Pollution card** (with offline fallback message)
- Added **Top Picks scope filter toggle**
- Added **Scope settings confirmation**
- Added **Target count summary**
- New **Image caching (scope-FOV only)** via Service Worker
- Added **version number banner**
